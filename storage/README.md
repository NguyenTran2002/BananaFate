# Storage

Storage layer interfaces for Google Cloud Storage and metadata database.

## Components

### GCS (`gcs/`)
- Google Cloud Storage utilities
- Signed URL generation for secure uploads
- Image lifecycle management
- Bucket configuration and permissions

## Features

- **Secure Upload**: Signed URLs with expiration
- **Cost Optimization**: Lifecycle policies for old data
- **Access Control**: Service account with minimal permissions
- **Metadata Tracking**: Integration with database layer
